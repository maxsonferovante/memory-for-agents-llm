# Cross-repo synthesis

Use this skill when the same concept appears in more than one repository.

## Output

- Shared invariant
- Repo-specific variations
- Ownership
- Canonical home
- Sync plan

## Constraints

- Classify facts by scope before writing anything.
- Keep the shared invariant in one place.
- Record the repo-specific delta separately.
