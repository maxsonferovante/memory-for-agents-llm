# Context pack

Use this skill when the main goal is to compress many sources into a short handoff.

For the live-session version of this workflow, use `.claude/templates/context-researcher-session.md`.

## Output

- Objective
- Canonical sources
- Verified facts
- Open questions
- Conflicts
- Relevant code paths
- Memory candidates

## Constraints

- Keep the pack small.
- Prefer source links over paraphrase.
- Separate shared facts from repo-local facts.
